# Phase 02 — Vertical MVP (shippable v0.1)

**Status:** ✅ Complete
**Targets:** `GrootKit` + `GrootApp` (SwiftUI app)

## Objective

Turn the Phase 01 runtime into a runnable, demoable macOS app with real file
monitoring, durable persistence, the first content-aware agent, live storage
agents, and the premium bubble/dashboard UI.

## Locked decisions

- App target generated by **XcodeGen** (`project.yml`) — not committed.
- Persistence via **system `libsqlite3`** (no external dependency).
- **Local-first AI**: Apple Vision OCR + heuristic naming; Ollama opt-in.
- Notarized/direct distribution (non-sandboxed); Full Disk Access via onboarding.

## Deliverables

### Runtime (`GrootKit`)
- **`SQLiteJournalStore`** — durable Undo journal (`~/Library/Application Support/Groot/groot.db`).
- **`VisionOCR`** + **`FilenameSuggester`** (`HeuristicFilenameSuggester`, optional `OllamaFilenameSuggester`).
- **`FSEventsWatcher`** + **`FileMonitoringAgent`** — watches Desktop/Downloads; pure flag→event mapping; loop-guarded via `.operationJournaled`.
- **`ScreenshotAgent`** — OCR → intelligent rename into `~/Pictures/Screenshots/YYYY-MM/`; preview/approval/autopilot.
- **`DownloadsOrganizerAgent`** — sorts new downloads into category folders (`FileCategory`).
- **`DesktopCleanerAgent`** — archives stale top-level Desktop files (intent + throttled tick).
- **`DuplicateDetectionAgent`** — SHA-256 grouping (size pre-filter, streaming hash); **approval-gated Trash, never auto-deletes**.
- **`StorageAnalyzerAgent`** — largest files + plain-language recommendations (read-only).
- **`ApprovingAgent`** protocol — UI routes approve/reject by `ApprovalRequest.agentID`.

### App (`GrootApp/`)
- `MenuBarExtra` + dashboard `Window`.
- Dashboard: stat tiles, scan actions (Scan Duplicates / Analyze Storage / Tidy Desktop), storage insights, duplicate report, approval cards, activity list with per-move **Undo**.
- Floating **bubble** panel — borderless non-activating `NSPanel`, spring/repulsion physics, glass tint per agent, drag + hover-expand.
- Full Disk Access onboarding banner.

## Verification

- `swift test` — 32 tests across all agents (SQLite durability, OCR suggester,
  FSEvents live delivery, screenshot pipeline, organizers, dedup approval→Trash, analyzer ranking).
- `xcodegen generate && xcodebuild -scheme GrootApp build` → BUILD SUCCEEDED; app launches.
